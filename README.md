# Asset Tracker API

REST API built with FastAPI for IT asset tracking, status updates, and user assignments.

---

## Tech Stack
*  🐍 **Python + FastAPI** — RESTful API development
*  🗄️ **MariaDB** — Relational database management
*  🐳 **Docker** — Containerization & deployment
